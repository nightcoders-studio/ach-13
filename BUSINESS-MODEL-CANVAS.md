## 12. Unit Economics

| Metrik | Nilai |
|--------|-------|
| Customer Acquisition Cost (CAC) | Rp 10.000 - 25.000 |
| Lifetime Value (LTV) - Free user | Rp 0 (indirect value via community) |
| Lifetime Value (LTV) - Premium | Rp 179.400 (avg 6 bulan × Rp 29.900) |
| Lifetime Value (LTV) - Les | Rp 600.000 (avg 3 bulan × Rp 200.000) |
| LTV/CAC Ratio (Premium) | 7.2x - 17.9x |
| Break-even | ~1.500 premium users atau 30 siswa les |
| Gross Margin (Premium) | ~85% (biaya utama = AI API) |
| Gross Margin (Les) | ~30% (70% ke tutor) |

## 13. Go-to-Market Strategy

### Phase 1: Soft Launch (Bulan 1-2)
- Deploy MVP (sudah live ✅)
- Invite 50 beta testers dari komunitas Aceh
- Collect feedback, iterate
- Seed konten awal (kosakata, lesson)

### Phase 2: Community Growth (Bulan 3-4)
- Launch di social media (Instagram, TikTok)
- "Kata Hari Ini" campaign
- Partnership dengan 2-3 komunitas Aceh perantauan
- Referral program (invite friend → bonus XP)

### Phase 3: Monetization (Bulan 5-6)
- Launch premium subscription
- Launch paket les (3 tutor awal)
- Approach Dinas Pendidikan untuk pilot program
- PR/media coverage tentang pelestarian bahasa

### Phase 4: Scale (Bulan 7-12)
- Mobile app launch
- B2B sales ke sekolah/kampus
- Expand content (multi-dialect)
- Hire community manager full-time
